# Skill: Exam Content Design

Use this skill when designing questions, exams, and topic metadata.

## Principles

- Questions should be tagged by subject, topic, skill, difficulty, year level, and exam type.
- Exams can be fixed or dynamic.
- V1 should prioritize fixed exams for predictable OC/Selective practice.
- Attempts must reference the exact question version used during the attempt.

## Quality Checks

Before publishing content, verify:

- Correct answer exists.
- Full explanation exists.
- Difficulty is reasonable.
- Topic tag is present.
- Exam type is present.
