# Skill: NSW Exam Domain

Use this skill when working on OC, Selective, NAPLAN, or HSC exam content.

## Rules

- Do not invent official exam timings or formats without verification.
- Treat OC and Selective as V1 scope.
- Treat NAPLAN as V2.
- Treat HSC as V3.
- Preserve exam type as explicit metadata.
- Preserve year level as explicit metadata.

## Content Requirements

Every published question needs:

- Correct answer
- Full explanation
- Subject
- Topic
- Difficulty
- Exam type
